# GT前端專案模板

整合常用基底Vite、TailwindCSS、SCSS、Auto Import等配置

## 使用
```shell=
npm i gt-template -g
```

## 建立專案
```shell=
npx gt-template@[版本號] [projectName] [template target]

// ex. npx gt-template@0.0.7-beta-2 Hello

```
### template target
目前只有一版，預設為/template內容
